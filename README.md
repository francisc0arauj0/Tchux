## About

Tchux is an operating system that I created as a hands-on learning project. Its main goal is to explore how modern operating systems work by building one from scratch.

## Building

To build Tchux it is strongly advised use [Tchux Build Tools](https://github.com/francisc0arauj0/Tchux/tree/main/txbt) and [Visual Studio Code](https://code.visualstudio.com/)
